# Storyboard Clarity Guide

How to read montages, flash cuts, and chapter order across all arc panel files. Use with [STORYBOARD-GUIDE.md](./STORYBOARD-GUIDE.md).

---

## Panel Tags (use on every ambiguous beat)

| Tag | Meaning | Visual cue for artists |
|-----|---------|------------------------|
| **PRESENT** | Main story timeline in this chapter | Full saturation; location matches chapter header |
| **FLASH — dojang** | Memory at Premier enrollment / training | Slight blur border, warmer or colder than present |
| **FLASH — tournament** | Memory at a past or parallel tournament | Clock or bracket visible if same venue |
| **FLASH inset** | Brief memory inside a present panel | Small panel or desaturated overlay |
| **ANCHOR** | Return shot that reorients reader (Ethan on bleachers, on Kwon's mat, etc.) | Same seat/pose as prior anchor panel |

**Rule:** If money, brochures, or enrollment checks appear → almost always **FLASH — dojang**, not tournament bleachers.

**Dialogue rule:** Off-panel "him" in VIP lines refers to the **paying kid in frame**, not Ethan, unless Ethan is in the same panel.

## Nightmare Montage Logic (Arc 1 Ch 1)

The opening nightmare is **one tournament day** (7:04 AM → 3:12 PM fight) with **memory inserts** explaining *why* Ethan is ignored.

```text
PRESENT anchor: Ethan alone on bleachers (clock advances)
  → FLASH dojang: VIP check + "We'll take good care of him" (another kid)
  → PRESENT: Ttong coaches favored kid at venue
  → PRESENT: Ethan's POV — far away
  → PRESENT: random parent stretch — Ttong doesn't look
  → FLASH dojang: scrimmage head kick — Ethan was best
  → PRESENT: 2:47 PM — alone — "Won every scrimmage"
  → PRESENT: staging → cards → KO bounty → "Go fight" → ring → kick → wake
```

**Thesis:** Same patch, different treatment. Skill ≠ attention. Money = care.

## Common Confusion Fixes (checklist for new panels)

1. Label **FLASH** vs **PRESENT** on every quick cut.
2. Name the **ANCHOR** location (bleachers, mat, hallway).
3. VIP / check / brochure → dojang unless explicitly at venue desk.
4. "We'll take good care of **him**" → favored kid visible or in next panel.
5. Montage panels share a **clock** or **phone time** when same day.
6. After FLASH block, one **PRESENT anchor** panel before next beat.
7. Chapter numbers in storyboard match [arcs/](../arcs/) — if arc duplicates a number, storyboard uses **Ch Xb** or timeline header.
