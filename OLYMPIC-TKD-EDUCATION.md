# Olympic Taekwondo — Sport Education Guide

Periodic **rules** and **technique** scenes for readers new to Olympic taekwondo sparring.  
Panels tagged `[TEACH]` in storyboard files match this schedule.

**Rule set:** WT/Olympic electronic scoring — **Paris 2024** values below. Regional/U20 in story uses **1:30 rounds**; Olympic/collegiate finals use **2:00**, best-of-3 **rounds** (score resets each round).

**Fight choreography:** [ACTION-CHOREOGRAPHY.md](./storyboards/ACTION-CHOREOGRAPHY.md) — panel templates, elite combos, expanded key fights.

---

## Scoring Quick Reference (In-World)

**Paris 2024 / WT Olympic (canonical):**

| Action | Points | Notes |
|--------|--------|-------|
| Punch to hogu | 1 | Manual — judges confirm |
| Body kick (regular) | 2 | Trunk PSS sensor |
| Head kick (regular) | 3 | Headgear PSS sensor |
| Turning body kick | 4 | PSS + 2 judges confirm rotation |
| Turning head kick | 5 | PSS + 2 judges confirm rotation |
| Opponent gam-jeom | +1 to you | Penalty point |

*Story shorthand (Kieryn):* **"Head is three. Turning adds two."**

**Legacy note:** Older drafts used "×2 multiplier" (6 pts turning head). Use **5** for all new storyboard/script work unless flashback is explicitly pre-2024.

**Advanced (introduced later):**
- Gam-jeom (penalty) — deducted from score
- Clinch / grab — not allowed; reset
- Protesting / VAR — mentioned at Korea level only

## Visual Teaching Panel Format

For `[TEACH]` panels, use when possible:

1. **Diagram overlay** — foot path arrow, strike zone highlighted on hogu
2. **MV or Kieryn as instructor** — Kieryn for sass, MV for fundamentals
3. **Slow-motion inset** — during matches for new readers
4. **Scoreboard insert** — numeric callout when point lands

**AI prompt add-on for teach panels:**
```
instructional martial arts diagram feel, labeled strike zones optional space for text,
clear leg chamber path, educational sports manga panel
```
